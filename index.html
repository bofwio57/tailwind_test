VSCODE 필수 확장 : Tailwind CSS IntelliSense

마켓플레이스에서 다운 받을 수 있으며, Tailwind 클래스 자동완성 기능
tailwind.config.js > 파일 만들기

tailwind.config.js

/** @type {import('tailwindcss').Config} */
module.exports = {
    content: [
        "./*.html", // 현재 폴더의 모든 html 파일 감지
        "./src/**/*.{html,js}", // 혹시 src 폴더가 있다면 그 안의 파일도 감지
    ],
    theme: {
        extend: {},
    },
    plugins: [],
};

장점/단점

장점 : 생산성의 극대화

HTML 내에서 바로 클래스를 추가하여 원하는 디자인을 즉시 구축 가능, 단순한 프로젝트에 적합

단점 : 가독성 저하

HTML 마크업이 복잡해 보이고 가독성이 저하될 수 있음

기본 클래스

단위체계

기본 단위(w-4)는 rem을 기반함
고정된 크기는 대괄호(w-[20px])를 사용하여 직접 지정 가능

클래스 사용 예제 코드

<!-- rem 기반 기본 단위 예시 (1rem = 16px) -->
<div class="w-4 h-4"></div> <!-- 16px * 16px 크기 -->
<div class="w-16 h-16 bg-blue-500"></div> <!-- 64px * 64px 크기 -->

<!-- 대괄호 문법을 활용한 임의의 고정값 (Arbitrary values) 지정 -->
<div class="w-[20px] h-[55px]"></div> <!-- 20px * 55px 고정 크기 -->

색상, 테두리, 라운드 처리

백그라운드 색상은 `bg-색상-숫자` 형태로 지정 숫자가 커질수록 색상의 명도와 채도가 짙어지는 구조

클래스 사용 예제 코드

<!-- 핑크 배경에 8px 검정색 테두리, 25px 라운딩 및 원형 실습 코드 -->
<!-- 1. 25px 라운드 처리된 핑크 박스 -->
<div class="bg-pink-400 border-8 border-black rounded-[25px] w-28 h-28 flex items-center justify-center">
  rounded-[25px]
</div>

<!-- 2. 완전히 둥근 원형 (rounded-full) -->
<div class="bg-pink-400 border-8 border-black rounded-full w-28 h-28 flex items-center justify-center text-xs font-bold">
  동그라미
</div>

레이아웃: 너비(W), 높이(H), 패딩(P), 마진(M)

width(W), height(H), padding(P), margin(M)
x축(좌우)과 y축(상하) 가능(mx-10), w-full > 화면 전체

텍스트

text-color, text-size, font-bold, cursor
cursor-pointer > 마우스 커서 모양을 변경

Flex

flex, flex-col, justify, items, gap-4
flex justify-center items-center > 정 가운데 정렬

Hover

hover:
hover:bg-purple-400 transition > 호버시 백그라운드가 보라색으로 변해라 자연스럽게(transition)

고급 기능: 그룹 호버(Group Hover)

group-hover > 최상위 요소에 부여
상위 요소에 마우스를 올렸을 때 하위 요소의 스타일이 동시에 변경
스타일 변경하는 하위 요소에 group-hover:~ 부여 하면 변경됨
